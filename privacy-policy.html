Privacy Policy

Last Updated: 2025-11-30

This Privacy Policy describes how IQ Quiz Pro ("the Application") handles data in connection with the personal use of this application for publishing content to TikTok via the TikTok API.

1. Application Overview

IQ Quiz Pro is a personal-use application operated solely by the application owner. The application is used exclusively to upload, schedule, and publish video content to TikTok through the TikTok API. There are no public users, sign-ups, or third-party access.

2. Information Collection

The application may collect and process the following types of information:

  • TikTok Account Data: Authentication tokens and account credentials required to access the TikTok API on behalf of the application owner.
  • Video Content: Video files and metadata (titles, descriptions, hashtags) prepared for upload to TikTok.
  • Analytics Data: Performance metrics and analytics retrieved from TikTok regarding published content.
  • Application Logs: Technical logs related to application operations, API requests, and error reporting.

3. How We Use Information

Information collected by the application is used solely for the following purposes:

  • Authenticating and maintaining connection to the TikTok API
  • Uploading and publishing video content to TikTok
  • Scheduling future content publications
  • Retrieving and analyzing content performance metrics
  • Troubleshooting technical issues and maintaining application functionality

4. Information Sharing and Third Parties

The application interacts with TikTok's services and is subject to TikTok's Privacy Policy and Terms of Service:

  • TikTok: Video content, metadata, and authentication credentials are transmitted to TikTok through their official API. TikTok's handling of this data is governed by their own privacy policy.
  • No Other Third Parties: The application does not share data with any other third-party services, advertisers, or external entities.

Please review TikTok's Privacy Policy at: https://www.tiktok.com/legal/privacy-policy

5. Data Storage and Security

Data security measures include:

  • Authentication tokens and credentials are stored securely and encrypted where possible
  • Access to the application and its data is restricted to the application owner only
  • Standard security practices are followed to protect against unauthorized access
  • Application data is stored locally or on secure servers under the owner's control

6. Data Retention

Data is retained as follows:

  • Authentication tokens are kept for as long as TikTok API access is required
  • Video content may be stored temporarily during the upload process
  • Analytics data may be retained indefinitely for performance tracking
  • Application logs may be retained for troubleshooting purposes

7. User Rights

As the application owner, you have full control over:

  • All data stored within the application
  • Revoking TikTok API access at any time through TikTok's authorization settings
  • Deleting any stored content, credentials, or application data
  • Discontinuing use of the application at any time

8. Children's Privacy

This application is not intended for use by children under 13 years of age. The application does not knowingly collect information from children.

9. International Data Transfers

When using TikTok's API, data may be transferred to and processed in countries where TikTok operates. These transfers are subject to TikTok's privacy practices and applicable data protection laws.

10. Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. The "Last Updated" date at the top of this page will reflect the most recent revision. Continued use of the application after changes constitutes acceptance of the updated policy.

11. Contact Information

For questions or concerns about this Privacy Policy or data handling practices, please contact the application owner.

12. Compliance

This application and its data handling practices are designed to comply with:

  • TikTok's API Terms of Service and Developer Policies
  • Applicable data protection and privacy regulations
  • General data security best practices for personal applications
